Graphait — opis designu
Ogólny styl
- Tło: bardzo jasne szare (#F8F9FA) lub czyste białe
- Akcent: jeden kolor — głęboki indygo (#4F46E5) lub slate blue
- Typografia: system-ui / Inter — duże, czytelne nagłówki, małe etykiety uppercase z letter-spacing
- Brak gradientów, brak ilustracji — tylko ikony (Lucide/Heroicons), linie, cienie max 1px
---
Ekrany
1. Login / Register
Centrum ekranu, karta 400px, logo + nazwa "Graphait" nad formularzem, dwa pola (email, hasło), jeden przycisk CTA. Minimalistycznie.
2. Agent Graph (główny ekran)
Lewy sidebar (240px) — lista agentów z avatarami (inicjały w kółku, kolor zależny od typu: indygo = AI, szary = human). Główna przestrzeń — force-directed graph z węzłami-kółkami, krawędzie to linie z etykietą "reports to". Kliknięcie węzła otwiera prawy panel (slide-in, 320px) with detalami agenta.
3. Task Board
Klasyczny kanban — 4 kolumny: Todo, In Progress, Waiting Approval, Done. Karty zadań: tytuł pogrubiony, assignee avatar, priorytet jako kolorowa kropka (red/orange/gray), numer zadania (#12) mały uppercase. Brak zbędnych ramek — tylko delikatne tło karty.
4. Task detail (modal / right panel)
Tytuł jako duży edytowalny h1, status dropdown, assignee picker, sekcja Comments — chronologicznie, autor + czas + treść, pole do wpisania nowego komentarza na dole.
5. Scheduler / Agent config (modal)
Formularz inline w panelu agenta — connector_type (select: http / opencode), interval_seconds (input number), toggle is_enabled. Proste, bez kroków.
